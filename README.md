# Métodos Numéricos

Repositorio académico de la materia de Métodos Numéricos.

## Objetivo

Documentar los temas vistos durante el curso mediante conceptos, fórmulas, pseudocódigos, ejercicios, resultados y códigos implementados.

## Temas del repositorio

| Tema                                                            | Descripción                                              |
| --------------------------------------------------------------- | -------------------------------------------------------- |
| [01 - Errores y aproximaciones](./01-errores-y-aproximaciones/) | Cálculo de errores absoluto, relativo y porcentual       |
| [02 - Raíces de ecuaciones](./02-raices-de-ecuaciones/)         | Métodos para encontrar raíces de funciones               |
| [03 - Sistemas de ecuaciones](./03-sistemas-de-ecuaciones/)     | Resolución de sistemas de ecuaciones lineales            |
| [04 - Interpolación](./04-interpolacion/)                       | Estimación de valores intermedios mediante interpolación |
| [05 - Integración numérica](./05-integracion-numerica/)         | Aproximación de integrales definidas                     |
| [06 - Ecuaciones diferenciales](./06-ecuaciones-diferenciales/) | Métodos numéricos para ecuaciones diferenciales          |

## Estructura de cada tema

Cada carpeta contiene los siguientes archivos:

| Archivo / Carpeta | Descripción                        |
| ----------------- | ---------------------------------- |
| README.md         | Explicación general del tema       |
| conceptos.md      | Conceptos, definiciones y fórmulas |
| pseudocodigo.md   | Algoritmos paso a paso             |
| ejercicios.md     | Problemas desarrollados            |
| resultados.md     | Resultados obtenidos               |
| codigos/          | Implementaciones en Python         |

## Herramientas utilizadas

* Python
* Git
* GitHub
* Visual Studio Code

## Contenido del repositorio

El repositorio incluye ejercicios prácticos, métodos numéricos básicos, algoritmos y programas desarrollados durante el curso con fines académicos.
