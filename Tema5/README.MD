# Tema 5 - Integración Numérica

## Descripción

La integración numérica permite aproximar el valor de una integral definida mediante procedimientos matemáticos iterativos cuando no es posible resolverla de forma analítica o cuando se busca una solución aproximada.

## Objetivo

Aplicar métodos numéricos para aproximar áreas bajo la curva utilizando fórmulas y algoritmos computacionales.
