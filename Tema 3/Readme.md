# Tema 3 - Sistemas de Ecuaciones

## Descripción

Los sistemas de ecuaciones lineales permiten resolver problemas matemáticos donde existen varias incógnitas relacionadas entre sí mediante diferentes ecuaciones.

## Objetivo

Aplicar métodos numéricos para resolver sistemas de ecuaciones lineales utilizando procedimientos algebraicos e iterativos.
