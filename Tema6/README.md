# Tema 6 - Ecuaciones Diferenciales

## Descripción

Las ecuaciones diferenciales permiten modelar fenómenos físicos, matemáticos e ingenieriles mediante relaciones entre funciones y sus derivadas.

## Objetivo

Aplicar métodos numéricos para aproximar soluciones de ecuaciones diferenciales ordinarias.

## Métodos implementados

* Método de Euler
* Método de Heun
* Método de Runge-Kutta de cuarto orden
* Método de Taylor de segundo orden
