# Tema 2 - Raíces de Ecuaciones

## Descripción

Las raíces de ecuaciones permiten encontrar los valores de una variable donde una función es igual a cero.
Los métodos numéricos de este tema se utilizan cuando no es posible resolver una ecuación de manera exacta mediante procedimientos algebraicos tradicionales.

## Objetivo

Aplicar métodos numéricos para aproximar raíces de funciones utilizando algoritmos iterativos y criterios de error.
